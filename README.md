# Valentine Hotline

A voice-based **Virtual Dating Assistant** — WebRTC rooms where suitors connect with AI voice agents (via LiveKit).

## Project structure

```
├── frontend/     # React SPA (Vite + TypeScript + LiveKit)
├── backend/      # (to be merged)
└── README.md
```

## Frontend

See [frontend/README.md](frontend/README.md) for setup, build, and deployment.

cd frontend && npm install && npm run dev
